Overview

Base class for all step handlers that defines the common interface and provides lifecycle event handling

Step handlers are responsible for executing the actual logic of workflow steps and reporting results.

def handle(task, sequence, step)
  # Fire the before_handle event
  Tasker::LifecycleEvents.fire(
    Tasker::LifecycleEvents::Events::Step::BEFORE_HANDLE,
    {
      task_id: task.task_id,
      step_id: step.workflow_step_id,
      step_name: step.name
    }
  )

  # Subclasses should implement the process method
  process(task, sequence, step)
end
