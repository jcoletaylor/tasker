Overview

Represents a relationship between two nodes in a dependency graph

Examples:

Basic edge

edge = GraphEdge.new(
  from: 'validate_payment',
  to: 'update_inventory',
  relationship: 'prerequisite'
)

Edge with metadata

edge = GraphEdge.new(
  from: 'step_1',
  to: 'step_2',
  relationship: 'prerequisite',
  metadata: { weight: 1.5, critical_path: true }
)
