Overview

Metadata about the overall dependency graph

Examples:

Template graph metadata

metadata = GraphMetadata.new(
  graph_type: 'template',
  handler_name: 'order_processing',
  total_nodes: 5,
  total_edges: 4
)

Runtime graph metadata

metadata = GraphMetadata.new(
  graph_type: 'runtime',
  task_id: '12345',
  total_nodes: 5,
  total_edges: 4,
  execution_context: {
    ready_steps: 2,
    blocked_steps: 1,
    completed_steps: 2
  }
)
